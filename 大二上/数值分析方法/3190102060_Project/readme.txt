程序运行说明：
需要使用程序时，请双击“Numerical Analysis .html”文件，通过浏览器打开。请注意，如果您的浏览器安全配
置较高，页面上方出现“为帮助保护你的安全，你的Web浏览器已经限制此文件显示……单击此处查看选项……”
提示，请单击它并选择“允许阻止的内容”，以保证程序功能的正常实现；除此之外，输入小孩信息时，请按照文
本框内的提示信息进行输入，确保格式以及内容的正确；输出估值顺序为-3SD，-2SD，-1SD，中位数，1SD，
2SD、3SD，也已提示在输出文本框中。当一次查询结束后，您可以对输出值进行复制等操作，也可以修改输入，
进行下一次查询。祝您使用愉快！

文件夹文件说明：
程序运行代码全部储存于“Numerical Analysis.html”文件当中，如果您需要查看，可以使用Visual Studio Code
打开文件。同时，为了更加方便地查看按钮响应主函数check()，特将其复制存放在了“project.js”文件中，双击
即可打开。“logo.jpg”用于页面图片呈现，为了确保页面显示正常，请将其与“Numerical Analysis.html”存
放在同一文件夹内。

关于月龄：
经查询，小孩的一个月龄为29.5天，因此81个月所对应的天数为2389.5天，而非2430天，请您在使用时确保输入
生日在程序有效查询范围之内（0-2389.5天），以避免输出报错，谢谢！
